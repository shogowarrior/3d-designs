# Summary

A pair of toy pliers made for my 3 year old son.  

# Print Settings

Printer: Monoprice Select Mini
Rafts: No
Supports: No
Resolution: 0.2mm
Infill: 30%

Notes: 
Print two copies of the STL.  I did not use a plastic pin, instead I used a small metric screw and a nylon nut with thread lock.
