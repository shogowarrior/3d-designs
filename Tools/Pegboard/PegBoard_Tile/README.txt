# Summary

This a PegBoard Tile that I created for an upcoming workbench project.  These will be combined to give me a nice peg board for the back of the bench.  I will also be designing some accessories and other tiles to go with the peg board.

Update:  I added bracing in the back of the tile, this is the PegBoard V2 Stl file.  I've added a frame for the tiles for extra gluing surface and reinforcements of the joints.
