# Summary

I added hooks to fit my board better.

Keep in mind that the width of the tray still doesn't quite allow for a zero horizontal separation between multiple trays. 

You can get away with smaller diameter pegs since they hook into the board.

The tray does not sit level with penomo's tray. The hooks must be near the top for easy peg insertion.

STEP file included
