# Summary

Very useful in helping to mount a box, the Shelves, canvas table frame
 ... easily and quickly !

# Instructions

Print the parts without support and position change. Put Fill density at 30%  and Fill pattern into Honeycomb  with a layer height of 0.3.  

you will also need:  

1 threaded rod M8 x 110mm  
3 nuts M8  
1 M3 x 15mm (or 20mm)  
1 M3 nuts

You can also put a [neodymium magnet][2] in the bottom of the hole in the slide piece so that it would retract better when the drive screw is loosened
