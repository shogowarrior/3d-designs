# Summary

Customisable holder for hex keys. Can be used for metric and inch sizes (both pictured); the comments in the `.scad` file explain how.

The holes should be slack except for an inner ring of 1.5mm that's a little too small that holds the key snugly. It might take some effort to chase this out the first time.
