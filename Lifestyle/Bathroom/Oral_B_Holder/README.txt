# Summary

This Oral B holder was initialy designed to be mounted on Wenko Vacuum Loc.
It is also now provided with a simple wall mount that can be fixed with screws.
It can hold an Oral B electric toothbrush and a spare brush, and can be used as a toothpaste holder/squeezer too.
The Oral B support (version 1& version 4 and more) was designed to be compatible with some other electric toothbrushes. 

UPDATE 2018-04-19
Added a wall mount part for using the holder without a Wenko Vacuum Loc.

UPDATE 2018-09-26
Added V2 with a more robust holder for the toothbrush body. This new holder is cleaner as the design allow more water evacuation but it's now only compatible with Oral B toothbrushes

UPDATE 2018-10-11
Added V4 now with 2 stronger and cleaner brush head holders. this version also brings back the compatibility with other electric toothbrushes

UPDATE 2018-11-14
Added V5 . Complete redesign to allow more water evacuation.
